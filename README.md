# Thunderbird Quick Steps

A Thunderbird addon that lets you create one-click buttons to run multiple email actions in sequence.

## Overview

Thunderbird Quick Steps is a Thunderbird add-on that lets you create custom quick-step buttons. Each quick step can run one or more actions on the currently selected message(s): copy, move, delete, archive, and more. You can create as many quick-step buttons as you want and chain multiple actions in a single quick step (for example: copy to Folder A, copy to Folder B, then move to Folder C).

## Key Features

- Create unlimited quick-step buttons.
- Configure each quick step to perform one or multiple actions in sequence.
- Supported actions: copy, move, delete, archive (and similar folder operations).
- Simple options UI to manage quick steps.
- Buttons available from the popup/options UI (see project files).

## Usage

1. After loading the add-on, open the Quick Steps popup or Options page to create and configure quick steps.
2. For each quick step, choose one or more actions.
3. Save the quick step — it will appear in the popup for quick access when handling messages.
